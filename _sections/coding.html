--- title: Software Development icon: fa-terminal order: 3 --- Python is my preferred language. But despite my love for it, it cannot be faster than a compiled, static typed language. So I find ways to make it work better than vanilla CPython, either by code acceleration (Cython, numba) or by multi-core computing (Multi-Threading/Processing, AsyncIO) or even different machines in a distributed computing network (Micro-Services, Apache Spark/Hadoop, Docker). Also more power can be squeezed out of the machine by programming GPUs (PyCUDA, numba.cuda) to perform computations. I also believe all code needs to be unit-tested (pytest, unittest) for its working and protection against regression.

--- title: Software Development icon: fa-terminal order: 3 --- Python is my preferred language. But despite my love for it, it cannot be faster than a compiled, static typed language. So I find ways to make it work better than vanilla either by code acceleration (ctypes, numba, C/C++ extensions) or by multi-core computing (Multi-Threading/Processing, AsyncIO) or even different machines in a distributed computing network (Pyro4, SCOOP, RPyC, micro-services, Apache Spark/Hadoop). Also more power can be squeezed out of the machine by programming GPUs (PyCUDA, numba.cuda). I also believe all code needs to be unit-tested for its working(pytest, unittest).
